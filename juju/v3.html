📚 Key Terms & Definitions

Function — A relation where every input (x) has exactly ONE output (y). Like a vending machine: one button = one item.

Not a Function — When the same input gives two different outputs. Example: (3,5) and (3,8) = NOT a function.

Linear — Makes a straight line. The rate of change is CONSTANT (same every time).

Non-Linear — Does NOT make a straight line. The rate of change is different each time.

Rate of Change — How much y changes when x increases by 1. Also called slope.

Domain — All the x-values (inputs) in a set of ordered pairs. List without repeats.

Range — All the y-values (outputs) in a set of ordered pairs. List without repeats.

Ordered Pair — A point written as (x, y). First number = x, second = y.

Independent Variable (x) — The variable YOU control. It does not depend on anything else. Usually time, items, days.

Dependent Variable (y) — The variable that CHANGES because of x. It depends on the independent variable.

Function Rule — An equation like y = mx + b that tells you how to calculate y from x.

Slope (m) — The rate of change. How steep the line is. In y=mx+b, m is the slope.

Y-Intercept (b) — Where the line crosses the y-axis. The starting value. In y=mx+b, b is the y-intercept.

Mean — The AVERAGE. Add all numbers and divide by how many there are.

Median — The MIDDLE number when sorted from smallest to biggest. If even count, average the two middle.

Mode — The MOST FREQUENT number. The one that appears the most. Can be none.

Range (Statistics) — The SPREAD of data. Biggest number minus smallest number.

Outlier — A number that is WAY far from all the others. Use median instead of mean when there is one.

Central Tendency — A single number that represents the center of the data: mean, median, or mode.

Coordinate Plane — The grid with x-axis (horizontal) and y-axis (vertical) crossing at the origin (0,0).

Quadrant I — Top right. Both x and y are positive (+,+).

Quadrant II — Top left. x is negative, y is positive (-,+).

Quadrant III — Bottom left. Both x and y are negative (-,-).

Quadrant IV — Bottom right. x is positive, y is negative (+,-).

Origin — The point (0,0) where the x-axis and y-axis cross.

Area of Triangle — A = ½ × base × height

Area of Rectangle — A = length × width

Perimeter of Rectangle — P = 2l + 2w (add all four sides)

Temperature: F to C — °C = (°F - 32) × 5/9

Temperature: C to F — °F = (°C × 9/5) + 32

Average Rate — Rate = Distance ÷ Time. Remember to convert minutes to hours first!
